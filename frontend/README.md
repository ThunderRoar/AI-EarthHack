## Getting Started
### Folder Structure
```yaml
.
├── public
├── src
├── .eslintrc.cjs
├── .gitignore
├── index.html
├── package.json
├── package-lock.json
├── README.md
└── vite.config.js
```
### Preparing Environment
1. `cd` into this directory and ensure that Node is installed
2. Run `npm install`
### Running the Frontend
1. Run `npm run dev`
2. Connect to the IP address in the console, it will likely be `http://localhost:5173/`
